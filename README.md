## Comandos de git:
- git add filename.ext ou -A -> add todos os arquivos alterados pra dentro do commit. -A add todos de uma vez
- git commit -m "message" -> commita os arquivos modificados, tenha cuidado pra não commitar o mesmo arquivo mais de uma vez para evitar conflito
- git push -u origin <nome da branch> -> tira do repositório local e joga para o github na branch desejada
- git push -u origin <nome da branch>:<outra branch> -> joga de uma branch para a outra. Verificar se é necessário merge manual no github
- git fetch para atualizar a branch atual
- git pull para baixar codigo da branch atual 
- git checkout para mudar de branch
- git checkout -b nome-da-branch para criar uma branch nova
